# leetcode　学習記録

4月から未経験エンジニアとして働くための準備と、Javaの基礎力を鍛えるためのリポジトリです。

##目標🎯
2026年4月の入社までにEASY問題を50問解く（または写経する）。
3年後(2029/4~）の独立に向けて、毎日コードに触れる習慣をつける。

##学習ルール📝
1．3分考えて分からなければ解説を見る。
2．解説を写経し、一行ずつ「なぜこの処理が必要か」コメントを入れる。
3．週末に三問だけざっと見直すor翌日に自力で再現できるか再挑戦する※2/5‐2/11までお試し期間
